# SpiceGarden — Platform APIs (Concept)

This document outlines key API domains. Implementation will be done in Phase 1.

## Auth
- Email login
- OTP/phone login
- Social login (Google/Apple)
- Forgot password
- Device/session management

## Customer
- Profile & preferences
- Saved addresses
- Favorites, order history
- Subscription membership and benefits

## Menu
- Categories
- Items
- Variants and add-ons
- Dynamic pricing and combo meals

## Orders
Supports:
- Instant
- Scheduled
- Group
- Corporate/Bulk
- Pickup

## Live Tracking
- Driver GPS updates
- ETA calculation
- Order status stream

## Payments
- Payment initiation
- Payment verification/webhooks
- Refund/credit handling

## Subscriptions
- Plan listing
- Subscription purchase
- Entitlements (free delivery, discounts, VIP support)

## Loyalty & Rewards
- Award points
- Redeem points/credits
- Cashback and referrals

## Support & Disputes
- Tickets
- Dispute resolution
- Auto refund rules (late/delay thresholds)
