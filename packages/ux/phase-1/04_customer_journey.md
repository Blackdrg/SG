# 04 — Customer Journey Flow (First-Time + Enterprise)

## Journey 1 — First-Time User
1. **Open app**
2. **Splash screen** (animated logo)
3. **Onboarding** (value proposition + quick steps)
4. **Choose location**
5. **Sign up**
6. **Permission requests**
   - Location
   - Notifications
   - Camera (only if needed; otherwise postpone)
7. **Home page**
8. **Search restaurant/food**
9. **Menu page**
10. **Add cart**
11. **Checkout**
12. **Payment**
13. **Live tracking**
14. **Delivery**
15. **Review**
16. **Retention offer** (next-order incentive)

### Prototype Requirements
- Each step must include at least:
  - loading state
  - error retry state

## Journey 2 — Fast Reorder
- Orders → select past order → confirm items → instant add to cart → checkout
- Goal: **reduce time-to-order under 15 seconds**.

## Journey 3 — Scheduled Order
- Cart → Schedule selection (date/time)
- Show availability constraints

## Journey 4 — Subscription Activation
- Subscription screen → select plan → payment → entitlements confirmation

## Journey 5 — Support / Dispute
- Orders → select order → support entry
- Guided form or quick options
